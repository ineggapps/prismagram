- [x] Create account
- [x] Requeset Secret (request key to log in with nodemailer)

- [x] Follow
- [x] Unfollow User
- [ ] Comment on a photo \*(I will skip delete, modify a comment)
- [ ] See user profile
- [ ] See the full photo
- [ ] Edit my profile
- [ ] Upload a photo
- [ ] Edit the photo (Delete)
- [ ] See the feed
